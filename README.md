# FastAPI Authentication System

A complete authentication and authorization backend built using FastAPI, PostgreSQL, SQLAlchemy, Alembic, and JWT.

---

## Features

- User Registration
- Login Authentication
- JWT Access Tokens
- Refresh Tokens
- Role-Based Access Control (RBAC)
- Email Verification
- Forgot Password
- Reset Password
- Protected Routes
- PostgreSQL Database
- Alembic Migrations

---

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT Authentication
- Passlib (bcrypt)
- Python

## Setup Instructions

### Clone Repository

```bash
git clone https://github.com/vasanthkumar597/Backend-foundation.git
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Server

```bash
uvicorn app.main:app --reload
```
